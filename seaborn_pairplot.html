- numpy - pandas - seaborn

Let's plot random numbers using numpy, pandas, and seaborn.

import numpy as np import pandas as pd import seaborn as sns sns.set_theme(style="ticks") x = np.random.randn(1000) y = np.random.randn(1000) z = np.random.randn(1000) c = ["A"] * 500 + ["B"] * 500 df = pd.DataFrame({"x": x, "y": y, "z": z, "color": c}) sns.pairplot(df, hue="color")
